$mfwiLinksProvider
service in module mfw-ionic.deep-links

Description

Provider of $mfwiLinks service.

Methods

  • addRoute(routeDefOrUrl, stateNameOrCallback)

    Declare a new route by setting the relative URI and an associated callback.

    Callbacks can be:

    • ui.router state name: this state will be accessed when the route is matched
    • An injectable function (both explicit and implicit).
      • You can inject an special object named $match where you'll receive the matched route.

    Example

    $mfwiLinksProvider
    // Route with data
    .addRoute({
        '/help': {
            callback: 'app.help',
            uiRouterParent: 'app.home'
        }
     })
    // Injectable callback
    .addRoute('/route/:param', ['$log', '$match', function ($log, $match) {
        $log.log('Matched route:', $match);
    }])
    // Route
    .addRoute('/settings', 'app.settings');
    Parameters
    ParamTypeDetails
    routeDefOrUrlstringobject

    Deep link route definition:

    • If string: URL to be matched.
    • If object: route definition. You can set an special key named uiRouterParent and $mfwiLinks will go to that parent state before accessing the target state.
    stateNameOrCallbackstringFunctionArray.<String|function()>

    Match callback:

    • If string: name of the ui-router state to be loaded.
    • If Function or Array.<String|function()>: injectable callback to be executed.
    Returns
    object

    Provider instance for nested calls.

  • config(options)

    Configure linking options.

    Example

        $mfwiLinksProvider.config({
        routesPrefix: '/app',
        matchCallback: successCallback,
        nomatchCallback: failCallback,
        nestedStatesDelay: 500
    });

    // Registering any route will have a '/app' prefix
    $mfwiLinksProvider.addRoute('/path', callback);

    // It's actually registered as `/app/path`
    // When a match is found, both `successCallback` and `callback` will be executed.
    Parameters
    ParamTypeDetails
    optionsobject

    Options

    PropertyTypeDetails
    matchCallback
    (optional)
    		Function

    Function to be executed for all matches.

    Defaults to: undefined

    nomatchCallback
    (optional)
    		Function

    Function to be executed for all non-matches.

    Defaults to: undefined

    routesPrefix
    (optional)
    		string

    Prefix to be prepended to all route calls.

    Defaults to: ""

    nestedStatesDelay
    (optional)
    		number

    Time to wait until load the accessed state when using parent state uiRouterParent reference.

    Defaults to: 800

    transitionHandlerFunctionArray.<String|function()>

    Injectable function that is responsible of matched transitions. Local injectable values:

    • $toState: name of the matched state
    • $toStateParams: object with matched parameters
    • $match: $match object

    Defaults to: $state.go($toState, $toStateParams);
